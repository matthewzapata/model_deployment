San Antonio Animal Shelter Project

A project with a goal of determining what factors make an animal more likely to be adopted.

This repo will contain python files and jupyter notebooks that analyze the data from three cities:

  • San Antonio
  • Lousiville
  • Bloomington

As stated above, the goal is to determine what factors play a role in how likely an animal is to be adopted. The result of this will allow one to determine which animals are likely to be adopted, and therefore determine which aren't as likely. More effort can be put into marketing these animals for adoption, finding a foster, or finding a rescue organization. The GitHub repository for this project can be found here.
